# Gestion des transactions dans un programme java

## **Objectifs du projet**
- Creer des comptes bancaires avec solde
- Effectuer des transactions (depot, retrait, virement)
- Enregistrer l'historique des transactions
- Afficher les informations des comptes et des transactions

## **Structure du projet**
- **Classe** Compte : represente un compte bancaire
- **Classe** Transaction : represente une operation (type, montant, date)
- **Classe** BanqueService :  contient la logique metier (depot, rerait, virement)
- **Classe** Main : programme principal pour tester
